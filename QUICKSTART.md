# MyKnowledge 快速入门（非技术用户版）

## 什么是 MyKnowledge？

简单来说，MyKnowledge 就像一个**智能笔记本**，帮你：
- 📁 自动整理项目资料
- 📝 记录要做的事情
- 📊 跟踪任务进度
- 🤖 自动帮你做记录

---

## 3 分钟上手

### 第一步：告诉 AI 你想做什么

直接说：
```
"我要开始一个新项目"
```

AI 会问你：
> 需要我为你创建知识库吗？
> - [全局知识库] - 放在你的个人文件夹，所有项目都能用
> - [项目知识库] - 放在当前项目文件夹，跟着项目走

**选哪个？**
- 如果只是个人笔记 → 选"全局"
- 如果是具体项目 → 选"项目"

### 第二步：让 AI 自动帮你记录

当你说：
```
"帮我分析一下这个销售数据，做个报表"
```

AI 会自动：
1. 发现这是一个复杂任务
2. 自动创建记录（不需要你手动操作）
3. 告诉你："已自动记录任务 REQ-001"

### 第三步：查看进度

随时问：
```
"项目进展如何？"
```

AI 会显示：
- 正在做的任务
- 已完成的任务
- 接下来该做什么

## 常见场景示例

### 场景 1：写报告

**你说**："我要写一份季度总结报告"

**AI 会做**：
- 自动创建记录
- 帮你分解步骤（收集数据→写大纲→填充内容→检查）

**AI 会做**：

### 场景 3：长期项目

**你说**："我要开发一个小程序"

**AI 会做**：
- 创建项目知识库
- 记录每个功能需求
- 跟踪开发进度
- 下次继续时不用重复说明

## 你可能遇到的问题

### Q: AI 什么情况下会自动创建记录？

**A**: 当你说的事情看起来比较复杂时，比如：
- 包含"分析"、"统计"、"设计"、"开发"等词
- 需要多个步骤完成
- 需要长期跟踪

**不想自动创建？** 直接说："不要自动记录"

### Q: 记录存在哪里？

**A**: 
- 全局记录：`你的个人文件夹/MyKnowledge/`
- 项目记录：`项目文件夹/.myknowledge/`（隐藏文件夹）

### Q: 怎么关闭自动记录？

**A**: 第一次出现时，AI 会问你"是否开启自动记录"，选"否"即可。

或者随时说：
```
"关闭自动记录功能"
```
