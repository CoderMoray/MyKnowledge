# 静默模式

当检测到复杂任务时，自动创建知识库。

> 💡 检测基于关键词规则，无法 100% 精准。如果触发频率不符合你的习惯，说"调整检测灵敏度"即可修改阈值。

---

## 复杂任务检测

### 触发条件

**关键词匹配**（满足 3 个及以上触发）：
- 分析、统计、挖掘
- 开发、设计、调研
- 整理、清洗、项目

**任务特征**：
- 多步骤操作
- 需要长期跟踪
- 涉及数据或文档

### 示例

| 用户输入 | 检测结果 |
|----------|----------|
| "帮我算 1+1" | ❌ 不触发 |
| "分析销售数据" | ✅ 触发 |
| "开发一个工具" | ✅ 触发 |

---

## 自动执行

```
1. 检测任务复杂度
2. 自动创建知识库（如不存在）
3. 自动创建需求记录
4. 告知用户已记录
```

---

## 用户控制

### 查看配置

从 `~/.myknowledge/config/skill-state.yaml` 读取 `auto_record` 设置。

### 临时禁用

用户可以说：
- "这次不要记录"
- "不要创建知识库"
- "简单处理"

### 完全关闭

修改 `settings.yaml`：
```yaml
features:
  silent_mode:
    enabled: false
```
